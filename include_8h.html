Detailed Description

Main include file.

This is a main include file for the entire ExternalMediaPackage project. It defines some important preprocessor variables that might have to be changed by the user.

Uncomment the define directives as appropriate

Francesco Casella, Christoph Richter, Roberto Bonifetto 2006-2012 Copyright Politecnico di Milano, TU Braunschweig, Politecnico di Torino

Macro Definition Documentation

◆ BUILD_DLL

#define BUILD_DLL   0

Build project into a DLL.

Set this preprocessor variable to 1 if the project is built into a dynamic link library. This setting influences the error reporting mechanism as well as the export statement.

◆ COOLPROP

#define COOLPROP   1

CoolProp solver.

Set this preprocessor variable to 1 to include the interface to the CoolProp solver developed and maintained by Ian Bell et al.
